☕ Starbucks Clone

Este projeto é uma recriação da interface do site da Starbucks com foco em prática de desenvolvimento front-end, responsividade e layout moderno.

📌 Sobre o projeto

A aplicação reproduz uma landing page da Starbucks, incluindo:
- Header com busca
- Banner principal
- Seções de conteúdo com grid layout
- Imagens estilizadas
- Layout responsivo

🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Google Fonts (Open Sans e Oswald)

🎨 Layout

O projeto utiliza:
- Flexbox (header)
- CSS Grid (seções)
- Media Queries para responsividade
- Background images para composição visual

📱 Responsividade

O site foi desenvolvido para funcionar em:
- 📱 Mobile
- 📲 Tablet
- 💻 Desktop

🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar:

- Estruturação semântica com HTML
- Estilização com CSS
- Grid Layout e Flexbox
- Responsividade
- Organização de código

⚠️ Aviso

Este projeto é apenas para fins educacionais e não possui qualquer vínculo com a Starbucks.
